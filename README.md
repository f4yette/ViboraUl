# ViboraUK

## HOW TO RUN WEBSITE LOCALLY


Follow steps below to run wesbite (server will be up and running using render and also use supabase for db)


---


## Install Node

Download LTS version:  
https://nodejs.org

Check installation:

```bash
node -v
npm -v
```

---

## Run Frontend

```bash
cd client
npm install
npm run dev
```

---

## Open In Browser

```
http://localhost:5173/
```
